# alpine-mariadb-client Docker image

This docker image contains:

- Alpine 3.8
- mariadb client only
- AWS CLI


## To build

```sh ./build.sh```

## To run

```sh ./run.sh```

## To shell into the running container
```sh ./exec.sh```

## To cleanup after running
```sh ./reset.sh```
